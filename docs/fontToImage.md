# 摘录卡片生成工具 使用说明

## 1. 功能简介

摘录卡片生成工具是一个支持多种美观背景、丰富字体、可自定义字号的文字图片生成器。你可以输入任意摘录内容，选择喜欢的背景和字体，调整字号，生成适合分享的精美图片。

---

## 2. 主要功能点

- **输入摘录内容**：支持多行文本输入。
- **背景选择**：支持纯色、渐变、格子、纹理等多种风格背景。
- **字体选择**：内置多款常用及设计感字体，支持中英文。
- **字号调整**：可通过按钮放大/缩小预览文字大小。
- **预览与滚动**：右侧预览区与输入区等高，内容超出时可滚动查看。
- **日期水印**：复制图片时自动添加当前日期，页面预览时不显示。
- **一键复制图片**：支持桌面端直接复制到剪贴板，移动端弹窗长按保存。
- **背景与字体色自动适配**：确保每种背景下文字都清晰美观。

---

## 3. 使用方法

1. **输入内容**  
   在左侧输入框输入你想生成图片的文字内容。

2. **选择背景**  
   在右侧"选择背景类型"与"选择背景"下拉框中，挑选你喜欢的背景风格。

3. **选择字体**  
   在"选择字体"下拉框中选择合适的字体，前排为设计感字体。

4. **调整字号**  
   点击"A-"或"A+"按钮，缩小或放大预览文字大小。

5. **预览效果**  
   右侧实时预览卡片效果，内容过多时可在卡片内滚动查看。

6. **复制图片**  
   点击"复制图片"按钮：
   - 桌面端：图片自动复制到剪贴板，可直接粘贴到微信、QQ等。
   - 移动端：弹窗显示图片，长按保存。

---

## 4. 常见问题与说明

- **为什么有的字体/背景在部分设备上显示不一样？**  
  设计感字体通过 Google Fonts 引入，部分字体需联网或特定系统支持。
- **为什么预览区有滚动条？**  
  当内容过多时，卡片内可滚动，保证图片生成时内容完整。
- **复制图片时日期会自动显示吗？**  
  是的，页面预览不显示日期，复制图片时自动加上当天日期水印。

---

## 5. 技术细节（开发者参考）

- 预览区与输入区通过 flex 布局等高。
- 背景与字体色适配通过 `bgTextColorMap` 自动切换，确保对比度。
- 复制图片用 `html2canvas`，截图前自动展开卡片、显示日期，截图后恢复原样。
- 字体通过 Google Fonts CDN 动态引入，支持多种设计感字体。
